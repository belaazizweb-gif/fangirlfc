## 2. Official star actions

Create a small controlled official star system.

Allowed actions for MVP:

```ts
publish_official_card: {
  stars: 0.5,
  xp: 50,
  once: true
}

replace_official_card: {
  stars: 0.25,
  xp: 25,
  once: false
}

share_official_card: {
  stars: 0.5,
  xp: 50,
  once: true
}

complete_profile: {
  stars: 0.5,
  xp: 50,
  once: true
}
```

Rules:

* once actions can only be rewarded once per user
* repeatable actions are allowed, but cap `replace_official_card` to max 3 rewards total
* never allow frontend to send arbitrary star values
* action names must be validated against the allowlist
