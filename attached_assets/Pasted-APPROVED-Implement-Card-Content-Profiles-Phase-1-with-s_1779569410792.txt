PHASE 5 — FIX FLAG ASPECT RATIO
==================================================

Current flags are stretched because flag zone aspect ratio differs from the real 4:3 flag ratio.

In FlagZone:
- pass contentProfile or relevant flag settings to FlagZone
- if contentProfile.preserveFlagAspectRatio is true, center a 4:3 flag inside the existing flag zone
- do not change flag assets
- do not change flag zone coords
- do not stretch the flag

Algorithm:
const targetRatio = contentProfile.flagAspectRatio; // 4/3
const zoneRatio = w / h;

if (zoneRatio > targetRatio) {
  flagH = h;
  flagW = h * targetRatio;
} else {
  flagW = w;
  flagH = w / targetRatio;
}

flagX = x + (w - flagW) / 2;
flagY = y + (h - flagH) / 2;

Use flagX/flagY/flagW/flagH for:
- image flag rendering
- fallback flag rectangle rendering

Important:
The fallback flag must also use corrected dimensions.
Do not correct only the image path and leave fallback stretched.
