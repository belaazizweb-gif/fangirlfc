ASSET RULES

For each uploaded template image:

1. Copy the original uploaded file to:

public/templates/{templateId}/source.png

2. Create overlay.png

For now, overlay.png can be the normalized version of source.png.

Do not redesign it.
Do not change colors.
Do not add text.
Do not add player photos.
Do not add logos.
Do not add stats.

3. Create background.png

For now, background.png should be a transparent 1086 × 1448 PNG.

Reason:
The current template images are mostly frame/overlay assets.
The user photo will be rendered behind overlay.png later.

4. Create thumbnail.png

Generate thumbnail.png from overlay.png.

Recommended size:
300 × 400 px or similar vertical ratio.

Keep aspect ratio.
Do not crop important parts.

DIMENSION RULES

Target size for overlay.png:

1086 × 1448

If source image is already 1086 × 1448, keep it.
If source image is not 1086 × 1448:
- normalize it to 1086 × 1448
- use contain-center on transparent background
- do not stretch badly
- keep source.png unchanged

Report original dimensions and final overlay dimensions.

TRANSPARENCY RULES

If image has real alpha transparency:
- preserve it

If image has fake checkerboard baked into the pixels:
- do not damage the template
- try safe cleanup only if reliable
- if not reliable, keep it as-is and report:
  “checkerboard appears baked, manual cleanup may be needed”
