B. Difficulty tuning
The game must not be too easy.
It should feel fair, skill-based, and slightly challenging.

Target balance:
- new users should often score 1/3 or 2/3
- 3/3 should feel rewarding, not common
- the game must not feel random or unfair

Please tune:
1. Perfect timing window
- make it slightly smaller so Perfect Shot is rewarding

2. Safe Shot vs Power Shot
- Safe Shot = easier to control, lower reward
- Power Shot = stronger / more dangerous, but riskier

3. Goalkeeper save logic
- if the keeper guesses correctly, average shots should often be saved
- perfect shots should be harder to save
- weak shots should be easier to save

4. 3rd penalty pressure
- make the 3rd shot slightly more intense
- slightly faster timing bar or slightly tighter timing window
- keep it subtle, not frustrating

5. Explain results clearly
- if missed: too much power / off target
- if saved: keeper read it
- if goal: clean finish / perfect shot / keeper wrong way
