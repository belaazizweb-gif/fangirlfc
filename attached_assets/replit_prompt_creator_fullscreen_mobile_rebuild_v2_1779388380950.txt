MAIN UX STRUCTURE

1. Fullscreen editor route

2. Top bar
- Center title: Créateur de cartes FC

3. Card area
- Card must be centered
- Card must be large
- Card should be the main visual element
- No template thumbnails above it
- No developer/debug content visible by default

4. Edit hotspots

Hotspots:
- photo zone
- rating
- position
- flag/country
- badge
- name
- stats
- optional template/card hotspot

Each hotspot must:

- not appear in exported PNG
