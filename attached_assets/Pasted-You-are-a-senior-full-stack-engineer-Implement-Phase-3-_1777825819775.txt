## 3. Team score calculation

MVP formula:

```ts
team.rankScore = totalXp + totalStars * 100
```

When a user earns official stars:

* read user profile
* get `officialTeamCode`
* if no officialTeamCode, skip team update
* if valid teamCode:

  * increment `teams/{teamCode}.totalStars` by the awarded stars delta
  * increment `teams/{teamCode}.totalXp` by the awarded XP delta
  * recompute/increment rankScore accordingly
  * ensure teamName is set

Use Firestore transaction if possible.

Do not recalculate all users.
MVP uses incremental aggregation.

---

## 4. Member count handling

For Phase 3, keep this simple and safe.

When a user sets official team for the first time:

* increment `teams/{teamCode}.memberCount` by 1

If user changes official team:

* decrement old team memberCount by 1
* increment new team memberCount by 1

Important:

* Do NOT move historical stars between teams in Phase 3.
* Only future stars go to the current official team.
* Add a code comment explaining this MVP decision.

If changing memberCount safely is too complex because team changes are handled in another function, create a helper:

```ts
setOfficialTeamWithAggregation(uid, newTeamCode)
```

And use it wherever official team changes are confirmed.
