==================================================

In removeBackgroundModnet.ts, update pipeline creation to try tasks in this order:

1. "background-removal"
2. "image-segmentation"

- Try "background-removal" first.
- If it throws during pipeline initialization, catch that initialization error.
- Then try "image-segmentation".
- If "image-segmentation" also throws, throw a new Error that includes both task names and both error messages.
- Do not silently ignore errors.
- Do not fallback to another model.
- Do not use RMBG.
- Do not fallback to @imgly inside this helper.
- Keep @imgly as the default engine outside the helper only.

      task name that worked

- active task used
- model id used
