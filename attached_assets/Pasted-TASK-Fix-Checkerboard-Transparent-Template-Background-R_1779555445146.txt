BACKDROP RULES

Add a backdrop before the full card base image.

Backdrop:
- x = 0
- y = 0
- width = CANVAS_WIDTH
- height = CANVAS_HEIGHT
- listening={false}

Prefer solid color first.
Do not add complex gradients unless already simple and safe.

Use per-template mapping:
