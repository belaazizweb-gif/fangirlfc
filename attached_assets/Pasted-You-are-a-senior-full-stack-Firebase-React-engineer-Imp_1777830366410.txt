## 2. Rival Detection (VERY IMPORTANT)

Find **closest competitor above user**:

```ts
rival = leaderboard[index - 1]
```

If exists, display:

```txt
You're just behind Ali (France)
```

Optional second line:

```txt
Only 0.5 ⭐ to pass them
```

Compute delta:

```ts
delta = rival.rankScore - user.rankScore
```

Convert to readable stars:

```ts
starsNeeded = ceil(delta / 100)
```
