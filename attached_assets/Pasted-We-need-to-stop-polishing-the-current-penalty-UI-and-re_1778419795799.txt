Gameplay flow per attempt:
1. User taps/selects a target inside the goal area.
2. User locks power/timing.
3. Ball animates from penalty spot toward the selected target.
4. Goalkeeper dives left/center/right or toward the selected zone.
5. Game determines:
   - GOAL
   - SAVED
   - MISSED
6. Show short feedback.
7. Move to next penalty until 3 attempts are completed.

Result logic:
