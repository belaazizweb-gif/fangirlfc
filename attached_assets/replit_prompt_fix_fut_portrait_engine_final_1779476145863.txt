Create a centralized portrait box helper.

lib/cardCreator/renderUtils.ts

Add:

getPortraitBox(layout, templateId): NormRect

Requirements:
- return normalized coordinates
- use the same default portrait box for all templates for now
- keep templateId as a parameter for future per-template tuning
- structure the function so future template-specific overrides are easy
- do NOT hardcode portrait coordinates inside CardCanvas.tsx

Default portrait box:

y: 0.10
w: 0.48
h: 0.43

Recommended structure:

type NormRect = {
  x: number;
  y: number;
  w: number;
  h: number;
};

const DEFAULT_PORTRAIT_BOX: NormRect = {
  x: 0.26,
  y: 0.10,
  w: 0.48,
  h: 0.43,
};

const PORTRAIT_OVERRIDES: Partial<Record<string, NormRect>> = {
  // Future examples:
  // gold_elite_2026: { x: 0.25, y: 0.10, w: 0.50, h: 0.44 },
};

export function getPortraitBox(layout: any, templateId: string): NormRect {
  return PORTRAIT_OVERRIDES[templateId] ?? DEFAULT_PORTRAIT_BOX;
}

Important:
Do not remove the layout parameter from the function signature. Keep it for future compatibility even if it is not used immediately.

NEW NO-PHOTO RENDERING

   pY = nY(portraitBox.y)
   pW = nW(portraitBox.w)
   pH = nH(portraitBox.h)

3. Render an optional subtle portrait backdrop:
   - no visible rectangular border
   - no obvious debug block
   - low opacity only
   - it must feel integrated into the card
   - if it looks like a rectangle, remove it

4. Render PLAYER_SILHOUETTE centered inside portraitBox.

Silhouette sizing:
- original SVG aspect ratio = 400 / 560
- silhouette height = portraitBox height × 0.92
- silhouette width = silhouette height × SVG aspect ratio
- centered horizontally and vertically
- opacity between 0.75 and 0.85
- listening={false}

Example rendering logic:

{!cardState.photo.src && silhouetteImage && (() => {
  const portraitBox = getPortraitBox(layout, templateId);

  const pX = nX(portraitBox.x);
  const pY = nY(portraitBox.y);
  const pW = nW(portraitBox.w);
  const pH = nH(portraitBox.h);

  const svgAspect = 400 / 560;
  const silH = pH * 0.92;
  const silW = silH * svgAspect;

  return (
    <KonvaImage
      image={silhouetteImage}
      x={pX + (pW - silW) / 2}
      y={pY + (pH - silH) / 2}
      width={silW}
      height={silH}
      opacity={0.82}
      listening={false}
    />
  );
})()}
